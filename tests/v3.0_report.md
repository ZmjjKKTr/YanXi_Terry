## 二、重要来电多轮对话 + 转接通知 (3/3)

### 2.1 医院急诊

```
[轮1] 来电: 你好这边是人民医院你母亲急诊住院了请尽快过来
      回复: 好的收到，情况我了解了。请问您是人民医院哪个科室的医生护士呀？

[轮2] 来电: 我是急诊科的王医生
      回复: 王医生您好，情况紧急我了解了。需要我马上记下什么或者帮您做什么吗？

[轮3] 来电: 病人需要家属签字才能手术请尽快过来
      回复: 好的，情况紧急我记下了，马上转告他。麻烦您留个科室的直接电话，让他马上过去直接联系您。

>>> 转接通知已打印，会话结束
```

### 2.2 辅导员开会

```
[轮1] 来电: 我是辅导员今天下午三点在学院会议室开会
      回复: 好的收到，通知他开会。请问您是哪个学院的哪位辅导员呀？

[轮2] 来电: 我是张老师
      回复: 好的，张老师您好。请问具体是要讨论什么事宜呢？我可以转告他。

[轮3] 来电: 讨论毕业答辩的事情挺重要的
      回复: 好的张老师，答辩的事我记下了，马上通知他。麻烦您留个联系方式，让他稍后马上联系您。

>>> 转接通知已打印，会话结束
```

### 2.3 HR面试通知

```
[轮1] 来电: 我是HR恭喜你通过了一面下周二上午十点二面
      回复: 好的收到，太好了，恭喜恭喜！请问是哪家公司的HR呀？

[轮2] 来电: 我是字节跳动HR部门的李经理
      回复: 好的，李经理您好。请问具体是什么岗位呀？我代他转告一下。

[轮3] 来电: 请带好简历提前十分钟到
      回复: 好的记下了，简历提前十分钟到。麻烦您留个联系方式，我好转告他。

>>> 转接通知已打印，会话结束
```

## 五、边界/复杂场景 (3/3)

| # | 场景 | 输入 | 回复 | 结果 |
|---|------|------|------|------|
| 1 | 非标准外卖表述 | 喂我给你放门口地上了啊 | 放门口可以，放东门快递柜吧，谢谢！ | ✅ |
| 2 | 含位置词非外卖(已知局限) | 你好我在东门等你半个小时了你怎么还没来 | 不好意思，我马上过去，麻烦放东门丰巢吧，谢谢！ | ⚠️ |
| 3 | 快递到付 | 你的快递到付的你现在方便下来拿吗 | 现在上课下不去，放东门快递柜吧，谢谢！ | ✅ |

> ⚠️ #2: "等你"场景被"东门"关键词误判为外卖。这是关键词系统的已知局限，后续可通过负向信号词（如"等你"→非外卖）优化。
